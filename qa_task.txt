Для работы с фреймворком выберите любой сайт для тестирования. На этом сайте выберите три любых страницы,
для ваших тестов, и для каждой страницы напишите как минимум любых 5 тест кейсов. После написания тест кейсов,
реализовать их в виде кода в вашем фрймворке.   

В фрейморке создать testng.xml в котором будут прогоняться все ваши созданные тесты.

Дополнительно приветствуется использование логирования, а также подключения какого-нибудь из 
фреймворков для создания отчета (например Allure).

По итогу у Вас должен быть готовый фрейм, в котором можно запустить все ваши тесты. Плюс
еще должно быть создано как минимум 15 тест кейсов. (тест кейсы можно писать в Excel, Word, либо
одной из TMS, например Test Rail).

Подготовить презентацию для демонстрации и защите вашего проекта.

P.S. удачи с написанием финального проекта. И не забываем про теорию. У нас это вся теория по ручному тестированию.
